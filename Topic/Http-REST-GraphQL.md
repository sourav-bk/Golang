HTTP (HyperText Transfer Protocol) is the foundation of data communication on the World Wide Web. It is an application-layer protocol and transport layer it used TCP protocol  for transferring web resources such as HTML documents, images, videos, APIs, etc., between a client (like a browser) and a server. 

its follows a client-server, request-response, and stateless communication model.

HTTP follows client-server model, with a client opening a connection to make a request, then waiting until it receives a response from the server. HTTP is a stateless protocol, meaning that the server does not keep any session data between two requests, although the later addition of cookies adds state to some client-server interactions.
